# Standardized Variant Reclassification Engine — dependencies
#
# The core engine, normalization, monitoring, validation harness, and most of the
# unit test suite use ONLY the Python standard library. There is nothing to
# install to run them.
#
# The storage layer (storage/, db/apply.py) and its PostgreSQL integration tests
# (tests/test_storage.py) DO require psycopg. API dependencies live in
# api/requirements.txt so the service layer can be installed independently.

# Required for the storage layer and tests/test_storage.py (they skip without it).
# Current checked PyPI version: psycopg-binary 3.3.4

# and continues if it is missing; the gate never depends on plotting.
