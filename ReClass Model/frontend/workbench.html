ReClass Evidence Workbench

Enter evidence → track coverage → triage curation → import variants

Session

Enter structured evidence

Persists a reviewer/pipeline-supplied criterion with provenance (source version, checksum, access date, reviewer, re-review date). Stored de-identified — keyed only on the public variant key.

Record coverage observation

The blocked / missing-category status is derived from the present criteria and the variant class.

Scan for curation gaps

Resolves a variant and surfaces unmatched/ambiguous identity, missing transcript context, missing cohort denominators, and pathogenic/benign conflicts. Resolution policy is handled downstream.

VCF / CSV variant import (dry-run)

Normalizes identities, detects duplicates, and previews evidence resolution. Nothing is persisted.
